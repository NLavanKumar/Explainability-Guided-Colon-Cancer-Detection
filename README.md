# Explainability-Guided Colon Cancer Detection

## 📌 Project Overview
This project focuses on detecting colon cancer using histopathological images with deep learning and explainable AI techniques.

## 🚀 Features
- EfficientNetB0-based classification
- Grad-CAM visualization for explainability
- 5-Fold Cross Validation
- GUI for image testing
- High accuracy performance

## 🧠 Model Details
- Backbone: EfficientNetB0
- Input Size: 224 × 224
- Output: Binary Classification (Cancer / Normal)

## 📊 Dataset
- LC25000 Dataset
- 10,000 colon images used
  - 5,000 Cancer
  - 5,000 Normal

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC Curve
- Precision-Recall Curve

## 🖥️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Gradio

## 💡 Explainability
Grad-CAM is used to highlight important regions in histopathological images that influence model predictions.

## 🧪 Results
- Achieved near-perfect classification performance
- Strong ROC and PR curves
- Consistent performance across 5 folds


## 📌 Future Work
- Testing on real clinical datasets
- Multi-class cancer classification
- Deployment as web application
